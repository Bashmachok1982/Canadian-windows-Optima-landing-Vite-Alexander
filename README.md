# 🇨🇦 Canadian Windows Optima — Landing Page

**Live Demo:**
https://bashmachok1982.github.io/Canadian-windows-Optima-landing-Vite-Alexander/

Это адаптивный лендинг, разработанный с использованием **Vite**, современного
сборщика, и чистого **HTML/CSS/JS** (vanilla).  
Цель — показать преимущества энергоэффективных окон и дверей, с реализацией
интерактивных секций и галерей.

---

## 📌 Функциональность

✔ Адаптивный дизайн (mobile / tablet / desktop)  
✔ Галерея окон с лайтбоксом (SimpleLightbox)  
✔ Разделы: преимущества, отзывы, технология, причины замены  
✔ Интерактивные эффекты hover/hover-scale на карточках  
✔ Форма заявки с валидацией  
✔ Анимации при скролле (AOS)  
✔ Интерактивный футер с социальными блоками  
✔ Мультиязычность (data-i18n)

---

## 🛠 Технологии

- 🧰 **Vite** — сборка проекта
- 💡 **Vanilla JS** — без фреймворков
- 📸 **SimpleLightbox** — лайтбокс для изображений
- 🎨 **CSS / Flexbox / Grid** — адаптивная верстка
- 🌀 **AOS** — анимации при прокрутке
